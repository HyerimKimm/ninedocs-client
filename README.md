# 프로젝트 실행하기

⚠️ CRA 시 React 버전과 React-testing 버전의 차이로 인해 프로젝트를 yarn으로 실행해야 합니다.

- `yarn install`
- `yarn start`
- `.env.development`에서 `REACT_APP_API_MOCK = 'true'`를 설정하면 로컬환경에서 네트워크 요청 없이 테스트할 수 있습니다.
